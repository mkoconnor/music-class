Harmonics Demonstration

What to try?

  • Press the start sound button.
  • Try entering 1/n, 1/n^2, 1/n^3 in the input box and see how the sound changes (press enter after entering each expression).
  • Try entering ifodd(n,1/n), ifodd(n,1/n^2), ifodd(n,1/n^3) in the input box and see how the sound changes.
  • Try moving the red dots manually and see how the sound changes.

More Information

Adjust the harmonics by dragging the points below, or by typing an expression in the input box and pressing enter.

The input box can take normal mathematical expressions. Additionally, you can enter ifodd(x,y), which will be y if x is odd and 0 otherwise.
